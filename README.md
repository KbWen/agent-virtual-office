# Agent Virtual Office

> AI Agent 團隊的辦公室模擬器 — 他們不只在跑 code，他們在**上班**。

## 專案結構

```
virtual-office/    # 主要應用程式（React + Vite）
deploy_brain.*     # AgentCortex 部署腳本（sh / ps1 / cmd）
```

## 快速開始

```bash
cd virtual-office
npm install
npm run dev
```

詳細說明請見 [virtual-office/README.md](virtual-office/README.md)。
