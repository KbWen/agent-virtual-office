Platform Launch Guide

Browser (Direct)

Full view or panel mode for embedding

http://localhost:5174 http://localhost:5174?mode=panel

Claude Code CLI

Open panel mode + URL params for initial state

open "http://localhost:5174?mode=panel&command=/implement&active=dev"

Claude Desktop

Embed as iframe. Passive title detection, no setup

<iframe src="http://localhost:5174?mode=panel">

Gemini CLI / Codex CLI

Panel mode + hash params, or use the Codex CLI helper script for file-backed status.

open "http://localhost:5174?mode=panel#dev=working&workflow=Task"

Codex App

Use this bridge UI or `bridge.js` from a host webview. Automatic host tool events still require a host-side emitter.

<script src="http://localhost:5174/bridge.js"></script> officeBridge.send({ source: 'codex-app', workflow: 'Reviewing PR', dev: 'working' })

Antigravity Panel

Embed panel mode, push via global variable

<iframe src="?mode=panel"> window.__office_status__ = { _seq: Date.now(), type: 'office-status', agents: [{role:'dev', status:'working'}] }

Any iframe / Artifact

postMessage from parent frame

iframe.contentWindow.postMessage({ type: 'office-status', agents: [{role:'dev', status:'working'}] }, '*')
